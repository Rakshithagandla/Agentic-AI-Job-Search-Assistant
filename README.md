# Agentic AI Job Search Assistant

## Overview

Agentic AI Job Search Assistant is an AI-powered system that analyzes resumes, extracts skills, recommends suitable job roles, identifies skill gaps, and provides career improvement suggestions using Google's Gemini AI.

## Features

- Resume Analysis
- Skill Extraction
- Job Role Recommendation
- Skill Gap Analysis
- Career Guidance
- Automatic PDF Report Generation

## Technologies Used

- Python
- Google Colab
- Gemini AI
- LangChain
- ReportLab
- GitHub

## Workflow

Resume Upload
→ Skill Extraction
→ Resume Analysis
→ Job Recommendations
→ Skill Gap Analysis
→ Career Suggestions
→ PDF Report Generation

## Output

The system generates:
- Resume Analysis Report
- Recommended Job Roles
- Missing Skills
- Career Roadmap
- PDF Report
